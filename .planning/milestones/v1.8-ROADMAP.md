**Status:** SHIPPED 2026-03-07

## Overview

## Milestone Summary

**Key Decisions:**
- Generator plugin convention: `{id}.generator.js` default export with id/name/category/roles/generate
- Scope builder returns typed scope objects (all/department/own) so generators are role-unaware
- Hours data sourced from pre-computed hours_summary collection
- Salary projection uses hardcoded MoE reference rates with classification x degree matrix
- Attendance queried from activity_attendance directly (not analytics service)
- Ministry audit delegates to exportService rather than reimplementing
- Excel shaper intentionally separate from contract.shapeResponse (EXPO-03)
- Used pdfkit for PDF generation (lightweight, no headless browser dependency)
- Reisinger-Yonatan TTF font for Hebrew RTL text rendering in PDFs
- 5 generator categories merged into 4 catalog categories (department+schedule combined)
- Custom HTML table for DefaultTableRenderer (existing Table component too opinionated)
- SVG gauge built without library (simple circle + stroke-dasharray/offset)

**Issues Resolved:**
- PDF Hebrew RTL text rendering (3 iterations to get word order correct)
- Export pagination cap (limit=99999 to get all rows through orchestrator pipeline)
- Percentage values divided by 100 for Excel numFmt compatibility

**Issues Deferred:**
- orchestraCount hardcoded to 0 in department-overview (no reliable orchestra-to-department mapping)

**Technical Debt Incurred:**
- orchestraCount in department overview set to 0 (by design, info-level)
