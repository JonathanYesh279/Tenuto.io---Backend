Add three new functions after `getTeacherRowDetails()` (after the closing brace around line 259), BEFORE the `TeacherFileStructureGuide` component:

**1. `formatStudentChange(change: any): string`**
- Takes a change object `{ field, oldValue, newValue }` from the backend
- Maps field paths to Hebrew labels using a `fieldLabels` Record:
  - `'academicInfo.class'` -> `'כיתה'`
  - `'academicInfo.studyYears'` -> `'שנות לימוד'`
  - `'academicInfo.extraHour'` -> `'שעה נוספת'`
  - `'academicInfo.lessonDuration'` -> `'זמן שיעור'`
  - `'academicInfo.isBagrutCandidate'` -> `'מגמת בגרות'`
  - `'academicInfo.instrumentProgress'` -> `'כלי נגינה'`
  - `'academicInfo.instrumentProgress[0].instrumentName'` -> `'כלי נגינה'`
  - `'academicInfo.instrumentProgress[0].ministryStageLevel'` -> `'שלב'`
  - `'academicInfo.instrumentProgress[0].currentStage'` -> `'שלב'`
  - `'personalInfo.age'` -> `'גיל'`
- For boolean `newValue`: return `${label}: ${newValue ? 'כן' : 'לא'}`
- When `oldValue` is non-null/non-undefined: return `${label}: ${oldValue} → ${newValue}` (use right arrow character)
- Otherwise: return `${label}: ${newValue}`
- Fallback: if field not in map, use the raw field string as the label

**2. `getTeacherMatchBadge(teacherMatch: any): React.ReactNode`**
- If `!teacherMatch || teacherMatch.status === 'none'`, return `null` (do NOT show badge for rows with no teacher data)
- `'resolved'`: green badge (`bg-green-100 text-green-700`) with CheckCircleIcon, text `מורה: {teacherMatch.teacherName}`
- `'unresolved'`: red badge (`bg-red-100 text-red-700`) with XCircleIcon, text `מורה לא נמצא`
- `'ambiguous'`: amber badge (`bg-amber-100 text-amber-700`) with WarningIcon, text `{teacherMatch.candidateCount} מורים אפשריים`
- Use same inline Tailwind span pattern as `getRowStatusBadge()`: `className="inline-flex items-center gap-1 px-2 py-0.5 rounded-full text-xs font-medium ..."`
- All icons at `size={12} weight="fill"`
- Default case returns `null`

**3. `getStudentRowDetails(row: any): React.ReactNode`**
Mirror the structure of `getTeacherRowDetails()`. Three branches:

*Error case:*
- If `row.error`, return `<span className="text-red-600">{row.error}</span>`

*Not-found (new student) case* (`row.status === 'not_found'`):
- Extract `mapped` via `row.data?.mapped || row.mapped || {}` (defensive, same as teacher pattern)
- Extract `teacherMatch` from `row.teacherMatch`
- Return a `<div className="space-y-1 text-xs">` containing:
  - `<div className="text-blue-600 font-medium">תלמיד חדש - ייווצר ברשומה חדשה</div>`
  - Conditionally show each mapped field (only when non-null/non-undefined/non-empty):
    - `mapped.instrument` -> `כלי נגינה: {value}`
    - `mapped.class` -> `כיתה: {value}`
    - `mapped.studyYears` (check `!= null`) -> `שנות לימוד: {value}`
    - `mapped.ministryStageLevel` -> `שלב: {value}`
    - `mapped.isBagrutCandidate` (check `!= null`) -> `מגמת בגרות: {value ? 'כן' : 'לא'}`
    - `mapped.lessonDuration` (check `!= null`) -> `זמן שיעור: {value}`
    - `mapped.extraHour` (check truthy) -> `שעה נוספת: {value}`
    - `mapped.departmentHint` -> `מחלקה: {value}`
  - At the end of the div: `{getTeacherMatchBadge(teacherMatch)}`
  - All info lines use `className="text-gray-600"`

*Matched (update) case* (`row.status === 'matched'`):
- Extract `mapped`, `teacherMatch`, `changes` (from `row.changes || []`)
- If no changes AND no meaningful teacherMatch (status is 'none' or undefined): return `<span className="text-gray-400">אין שינויים</span>`
- Otherwise return `<div className="space-y-1 text-xs">` containing:
  - Changes mapped via `changes.map((change, idx) => <div key={idx} className="text-gray-700">{formatStudentChange(change)}</div>)`
  - At the end: `{getTeacherMatchBadge(teacherMatch)}`

Return `null` for any other case.
