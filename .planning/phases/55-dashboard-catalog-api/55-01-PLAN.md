<task type="auto">
  <name>Task 1: Create dashboard service with KPI computation, trends, and alerts</name>
  <files>api/reports/report.dashboard.js</files>
  <action>
Create `report.dashboard.js` exporting `buildDashboard(scope, { services })`.

**KPI Cards** (6 cards, each with `{ id, label, value, unit, trend, drillTo }`):

1. **activeStudents** — count of `student` docs with `{ tenantId, isActive: true }`. Label: `תלמידים פעילים`. Unit: `count`. drillTo: `student-enrollment`.
2. **activeTeachers** — count of `teacher` docs with `{ tenantId, isActive: true }`. Label: `מורים פעילים`. Unit: `count`. drillTo: `teacher-roster`.
3. **totalWeeklyHours** — sum of `totals.totalWeeklyHours` from `hours_summary` docs with `{ tenantId, schoolYearId }`. Label: `שעות שבועיות`. Unit: `hours`. drillTo: `teacher-hours-summary`.
4. **orchestraCount** — count of `orchestra` docs with `{ tenantId, schoolYearId }`. Label: `הרכבים`. Unit: `count`. drillTo: `orchestra-participation`.
5. **assignmentRate** — percentage of active students that have at least one entry in `teacherAssignments`. Label: `אחוז שיבוץ`. Unit: `percentage`. drillTo: `student-teacher-assignments`.
6. **dataQualityScore** — 100 minus (number of high-severity anomalies), floored at 0. Count anomalies: students without assignments + teachers without students (same logic as data-quality generator but just counts). Label: `ציון איכות נתונים`. Unit: `score`. drillTo: `data-quality`.

**Trend Calculation:**
- Accept `schoolYearId` (current) from scope context. Query the `school_year` collection for the current tenant to find the previous year (sort by `createdAt` desc, pick the one after the current). If no previous year, trend is `null`.
- For each KPI, compute the same metric for the previous year. Trend object: `{ delta, direction }` where `delta` = current - previous as a percentage change (rounded to 1 decimal), and `direction` = `'up'` | `'down'` | `'stable'` (stable if |delta| < 1%).

**Alerts** (array of `{ id, type, severity, message, drillTo }`):
- `idle-teachers`: If any teacher has no assigned students (count > 0). severity: `warning`. message: `{count} מורים ללא תלמידים`. drillTo: `data-quality`.
- `unassigned-students`: If any student has no teacherAssignments (count > 0). severity: `warning`. message: `{count} תלמידים ללא שיבוץ`. drillTo: `data-quality`.
- `stale-imports`: Query `import_log` for tenant, find the most recent import by `createdAt`. If older than 30 days or no imports exist, severity: `info`. message: `לא בוצע ייבוא נתונים מעל 30 יום`. drillTo: `import-history`.
- `low-data-quality`: If dataQualityScore < 80. severity: `warning`. message: `ציון איכות נתונים נמוך: {score}`. drillTo: `data-quality`.

Only include alerts where the condition is true (non-empty array).

Use `getCollection` from services (same pattern as generators). Accept scope with `tenantId` and `schoolYearId`. For 'own' scope, return empty KPIs and no alerts (dashboard is admin-only but handle gracefully).

Queries should be efficient: use `countDocuments` where possible, aggregate only when needed. Run independent queries in parallel with `Promise.all`.
  </action>

  </verify>
