**2. `detectEnsembleColumns(headers)` function** — Iterates over the `headers` array returned by `parseExcelBufferWithHeaderDetection`. For each header, strip Unicode direction markers (`[\u200F\u200E\uFEFF\u200B]`), then check if the cleaned header exists in `ENSEMBLE_COLUMN_HEADERS`. Returns an array of `{ index, header, subType }` objects. This follows the same pattern as `detectInstrumentColumns`.

**3. `normalizeOrchestraName(name)` function** — Trims whitespace, normalizes geresh variants (`[׳'` + backtick + right single quote]` to a single apostrophe `'`), and collapses multiple spaces. Returns the normalized string.

**4. `buildOrchestraNameIndex(orchestras)` function** — Takes an array of orchestra documents. Builds a Map where key = `normalizeOrchestraName(orch.name)` and value = array of orchestra docs with that normalized name. This handles the edge case of multiple orchestras sharing a name (disambiguated by subType).

**5. `readEnsembleNames(row, ensembleColumns)` function** — For each ensemble column, reads the cell value from `row[col.header]`. Skips empty/falsy/non-string values. Splits on `','` (comma separator — the export concatenates multiple orchestra names with `", "`, see ministry-mappers.js line 240-241). Trims each name. Returns array of `{ name, columnHeader, expectedSubType }`.

**6. `matchOrchestraByName(importedName, nameIndex, expectedSubType)` function** — Normalizes the imported name, looks up in the nameIndex Map. If no candidates: return `{ status: 'unresolved', importedName }`. If one candidate: return `{ status: 'resolved', orchestraId: candidates[0]._id.toString(), orchestraName: candidates[0].name }`. If multiple candidates: use `expectedSubType` to disambiguate — find candidate where `orch.subType === expectedSubType`, and return it as resolved. If no subType match, return the first candidate as resolved (still a valid match).

```javascript
// Bulk enroll students in orchestras (dual $addToSet — Pitfall 3 from research)
let orchestraLinkCount = 0;
if (orchestraStudentLinks.length > 0) {
  try {
    const orchestraCollection = await getCollection('orchestra');
    // Note: studentCollection already declared at top of function

    // Bulk update orchestra.memberIds
    const orchestraBulkOps = orchestraStudentLinks.map(({ orchestraId, studentId }) => ({
      updateOne: {
        filter: { _id: ObjectId.createFromHexString(orchestraId), tenantId },
        update: { $addToSet: { memberIds: studentId } },
      },
    }));
    await orchestraCollection.bulkWrite(orchestraBulkOps, { ordered: false });

    // Bulk update student.enrollments.orchestraIds
    const studentBulkOps = orchestraStudentLinks.map(({ orchestraId, studentId }) => ({

        filter: { _id: ObjectId.createFromHexString(studentId), tenantId },
        update: { $addToSet: { 'enrollments.orchestraIds': orchestraId } },
      },
    }));
    await studentCollection.bulkWrite(studentBulkOps, { ordered: false });

    orchestraLinkCount = orchestraStudentLinks.length;
  } catch (err) {
    console.error('Orchestra enrollment bulk write failed:', err.message);
    // Non-fatal: student import succeeded, orchestra linking failed

    errors.push({ error: `שגיאה בשיוך תלמידים להרכבים: ${err.message}` });

}
```
