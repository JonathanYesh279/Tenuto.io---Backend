**1. In `src/services/apiService.js`**, add an `attendanceAlerts` namespace to the apiService object (alongside existing namespaces like `students`, `teachers`, etc.):

```js
attendanceAlerts: {
  getSettings: () => fetchWithAuth('/api/attendance-alerts/settings'),
  getFlaggedStudents: (orchestraId) => fetchWithAuth(`/api/attendance-alerts/orchestra/${orchestraId}/flagged`),
  getDashboard: (params = {}) => {
    const query = new URLSearchParams();
    if (params.startDate) query.set('startDate', params.startDate);
    if (params.endDate) query.set('endDate', params.endDate);
    const qs = query.toString();
    return fetchWithAuth(`/api/attendance-alerts/dashboard${qs ? '?' + qs : ''}`);
  },
  getStudentSummary: (studentId) => fetchWithAuth(`/api/attendance-alerts/student/${studentId}/summary`),
},
```
