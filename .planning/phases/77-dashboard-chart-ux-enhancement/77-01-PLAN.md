<objective>
Create responsive dashboard chart section — tabbed interface on mobile, grid with focus-mode expansion on desktop.

Purpose: On mobile, the 3-column chart grid squishes charts to unusable widths. This replaces it with a tab-per-chart layout using the existing animated-tabs component. On desktop, charts stay in a 3-column grid but gain a focus-mode where clicking a chart header expands it full-width for deeper analysis.

Output: useMediaQuery hook, DashboardChartSection component, updated Dashboard.tsx
</objective>

**1. Create `src/hooks/useMediaQuery.ts`:**
- Generic hook: `useMediaQuery(query: string): boolean`
- Uses `window.matchMedia` with `addEventListener('change', handler)` for reactive updates
- SSR-safe: initialize with `typeof window !== 'undefined' ? window.matchMedia(query).matches : false`
- Clean up listener on unmount/query change
