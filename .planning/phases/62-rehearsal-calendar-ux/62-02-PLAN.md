**In rehearsalUtils.ts**, add a `detectCalendarConflicts` function:

```typescript
export function detectCalendarConflicts(rehearsals: Rehearsal[]): Map<string, ConflictInfo[]>
```

This function takes ALL rehearsals displayed on the calendar and returns a Map keyed by rehearsal._id, where the value is an array of conflict details. Two rehearsals conflict if:
- They are on the same date (compare date strings after normalizing to YYYY-MM-DD)
- Their time ranges overlap (startTime/endTime string comparison: A.startTime < B.endTime && B.startTime < A.endTime)
- AND they share either the same `location` (room conflict) OR the same conductor (teacher conflict — check `rehearsal.orchestra?.conductor?._id`)

ConflictInfo type:
```typescript
export interface ConflictInfo {
  type: 'room' | 'teacher'
  conflictingRehearsalId: string
  conflictingName: string  // orchestra name of the conflicting rehearsal
  detail: string  // e.g. "חדר: אולם מוזיקה" or "מנצח: יונתן"
}
```
