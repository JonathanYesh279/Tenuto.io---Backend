- For each background cell, if the slot index is NOT in the occupied set, add `onClick` handler and `cursor-pointer hover:bg-gray-50` class.

Implementation:
```typescript
// Inside the room row rendering, before the TIME_SLOTS.map:
const occupiedSlots = new Set<number>()
for (const activity of room.activities) {
  const startMin = timeToMinutes(activity.startTime)
  const endMin = timeToMinutes(activity.endTime)
  const gridStartMin = GRID_START_HOUR * 60
  for (let t = startMin; t < endMin; t += SLOT_DURATION) {
    const idx = Math.floor((t - gridStartMin) / SLOT_DURATION)
    if (idx >= 0 && idx < TIME_SLOTS.length) occupiedSlots.add(idx)
  }
}

// Then in the TIME_SLOTS.map for background cells:
```
