```javascript
return teachers.map(t => ({
  ...t,
  studentCount: countMap.get(t._id.toString()) || 0,
  loginCount: t.credentials?.loginCount || 0,
  lastLogin: t.credentials?.lastLogin || null,
```

This is a simple passthrough — the field already exists on the teacher document (once populated by import or calculation). No additional DB query needed. If the teacher has never had hours calculated, it returns `null`.
