  <files>api/import/import.service.js</files>
  <action>
  Add the following to `api/import/import.service.js`:

  1. **Add `previewEnsembleImport` to the `importService` export object** (alongside the existing previewTeacherImport, previewStudentImport, previewConservatoryImport).

  2. **`buildConductorCache(teachers)`** — Returns a function `matchConductor(nameString)`:
     - Maintains an internal `Map` keyed by `nameString.trim()`
     - On first encounter of a name: calls the existing `matchTeacherByName(nameString, teachers)` and caches the result
     - On subsequent encounters: returns cached result
     - For empty/null names: returns `{ status: 'none' }` (no cache needed)
     - Place this function near the existing `matchTeacherByName` function (around line 1310)

  3. **`matchExistingOrchestras(parsedRows, existingOrchestras)`** — For each parsed row with a resolved conductor:
     - Match criteria (per locked decision): **exact name + exact conductorId**. Both must match.
     - `existingOrchestras` is an array of orchestra documents from the DB
     - For each parsed row, find an existing orchestra where `orchestra.name === parsedRow.name` AND `orchestra.conductorId === parsedRow.conductorMatch.teacherId`
     - If matched: determine if there are changes by comparing type, subType, performanceLevel, schedule, and hours fields. Return status `'existing-updated'` if any differ, `'existing-no-change'` if identical.
     - If no match: status is `'new'`
     - Return a `Map<rowIndex, { status, orchestraId?, orchestraName? }>`

  4. **`async function previewEnsembleImport(buffer, options = {})`** — The main preview function:

     **Step 1:** Extract tenantId via `requireTenantId(options.context?.tenantId)` and schoolYearId from `options.context?.schoolYearId`.

     **Step 2:** Call `parseEnsembleSheet(buffer)`. If null or error, throw appropriate error ('גליון "הרכבי ביצוע" לא נמצא בקובץ' for null, or the specific missing_columns error).

     **Step 3:** Fetch all teachers for this tenant (for conductor matching):
     ```javascript
     const teacherCollection = await getCollection('teacher');
     const teachers = await teacherCollection.find(
       { tenantId, isActive: { $ne: false } },
       { projection: { 'personalInfo.firstName': 1, 'personalInfo.lastName': 1 } }
     ).toArray();
     ```

     **Step 4:** Build conductor cache and match all parsed rows:
     ```javascript
     const matchConductor = buildConductorCache(teachers);
     for (const row of parsedResult.parsedRows) {
       row.conductorMatch = matchConductor(row.conductorName);
     }
     ```

     **Step 5:** Fetch existing orchestras for this tenant+schoolYear (for orchestra matching):
     ```javascript
     const orchestraCollection = await getCollection('orchestra');
     const existingOrchestras = await orchestraCollection.find(
       { tenantId, schoolYearId, isActive: { $ne: false } },
       { projection: { name: 1, conductorId: 1, type: 1, subType: 1, performanceLevel: 1, scheduleSlots: 1, ministryData: 1, memberIds: 1 } }
     ).toArray();
     ```

     **Step 6:** Match existing orchestras using `matchExistingOrchestras()`. Attach `orchestraMatch` to each parsed row.

     **Step 7:** Build summary statistics:
     ```javascript
     const conductorSummary = { resolved: 0, unresolved: 0, ambiguous: 0, none: 0 };
     const orchestraMatchSummary = { new: 0, existingUpdated: 0, existingNoChange: 0 };
     // Count from parsed rows
     ```

     **Step 8:** Assemble the ensemble entries for the preview response. Each entry:
     ```javascript
     {
       row: parsedRow.row,
       rawName: parsedRow.rawName,
       name: parsedRow.name,
       type: parsedRow.type,
       subType: parsedRow.subType,
       performanceLevel: parsedRow.performanceLevel,
       participantCount: parsedRow.participantCount,
       conductorMatch: parsedRow.conductorMatch, // { status, teacherId?, teacherName?, candidateCount?, importedName }
       schedule: parsedRow.schedule,             // { activity1, activity2 }
       hours: parsedRow.hours,                   // { totalActual, coordinationHours, totalReporting }
       ministryUseCode: parsedRow.ministryUseCode,
       orchestraMatch: parsedRow.orchestraMatch, // { status, orchestraId?, orchestraName? }
     }
     ```

     **Step 9:** Build preview object:
     ```javascript
     const preview = {
       totalRows: parsedResult.parsedRows.length,
       ensembles: ensembleEntries,
       conductorSummary,
       orchestraMatchSummary,
       analytics: parsedResult.analytics,
       warnings: parsedResult.warnings,
       errors: [],
     };
     ```

     **Step 10:** Save to import_log (follow exact conservatory import pattern):
     ```javascript
     const importLogCollection = await getCollection('import_log');
     const logEntry = {
       importType: 'ensembles',
       tenantId,
       schoolYearId: schoolYearId || null,
       status: 'pending',
       createdAt: new Date(),
       preview,
       parsedData: {
         rows: parsedResult.parsedRows,
         analytics: parsedResult.analytics,
       },
     };
     const result = await importLogCollection.insertOne(logEntry);
     return { importLogId: result.insertedId.toString(), preview };
     ```

  5. **Wire 'ensembles' into `executeImport()` dispatcher** — Add an `else if (log.importType === 'ensembles')` branch that throws `new Error('ייבוא הרכבים יתמך בגרסה הבאה')`. This is a placeholder for Phase 24.
