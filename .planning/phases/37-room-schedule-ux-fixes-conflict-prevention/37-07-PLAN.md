**Fix 2: Teacher conflict check ID comparison (lines 118-131)**

The teacher conflict check at line 122 does:
```tsx
activity.teacherId === selectedTeacherId
```

Both values should be strings, but `activity.teacherId` comes from the schedule API response (which returns MongoDB ObjectId as string) and `selectedTeacherId` comes from `teacher._id` (which also comes from the API as string). The comparison should work, but add a `String()` coercion to be safe:

```tsx
if (
  String(activity.teacherId) === String(selectedTeacherId) &&
  doTimesOverlap(state.startTime, endTime, activity.startTime, activity.endTime)
)
```

Also, the current teacher conflict check only runs when a teacher is selected (`if (selectedTeacherId)`). This is correct. But it should also skip the current room's conflict check for teacher double-booking (to avoid counting the same room conflict twice). The room conflict is already reported separately. To avoid duplicate warnings for the same activity:

In the teacher double-booking loop (lines 119-130), add a skip for the activity that already triggered a room conflict. Actually, looking at the code more carefully: the room conflict checks activities in `state.room`, and the teacher conflict loops ALL rooms. So an activity in the target room at the target time would be reported BOTH as a room conflict AND a teacher conflict. This is actually correct behavior -- it IS both a room conflict and a teacher double-booking. But to avoid confusing duplicate messages, skip activities in the target room during the teacher check:

```tsx
if (selectedTeacherId) {
  for (const room of scheduleData.rooms) {
    if (room.room === state.room) continue  // already reported as room conflict
    for (const activity of room.activities) {
      if (
        String(activity.teacherId) === String(selectedTeacherId) &&
        doTimesOverlap(state.startTime, endTime, activity.startTime, activity.endTime)
      ) {
        // ... existing warning push
      }
    }
  }
}
```

This way the teacher conflict only fires for OTHER rooms where the teacher is already scheduled.
