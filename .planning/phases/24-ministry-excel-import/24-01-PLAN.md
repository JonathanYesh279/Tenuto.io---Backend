  <name>Task 1: Implement executeEnsembleImport with bulk operations and buildScheduleSlots helper</name>
  <files>api/import/import.service.js</files>
  <action>
Add two new functions to `api/import/import.service.js`:

**1. `buildScheduleSlots(schedule)` helper** (place near the ensemble section, after the existing ensemble helpers around line ~2835):
- Accepts a `schedule` object with optional `activity1` and `activity2` sub-objects (each having `dayOfWeek`, `startTime`, `endTime`)
- Returns an array of schedule slot objects matching the `scheduleSlotSchema` from `orchestra.validation.js`
- Only include a slot if `dayOfWeek` is not null/undefined
- Default startTime/endTime to `'00:00'` if missing

**2. `executeEnsembleImport(log, importLogCollection, tenantId, userId)` function** (place after `executeConservatoryImport`, before the preview functions section):

The function must:

a) **Add import for validateOrchestra** at the top of the file (add to imports):
   ```javascript
   import { validateOrchestra } from '../orchestra/orchestra.validation.js';
   ```

b) **Get collections:** `orchestra` and `teacher` via `getCollection()`

c) **Resolve schoolYearId:** Use `log.schoolYearId` from the import_log entry (saved during preview at line 2822). If null, dynamically import `schoolYearService` and call `getCurrentSchoolYear({ context: { tenantId } })`. If still no school year, throw error: `'לא נמצאה שנת לימודים - לא ניתן לבצע ייבוא'`

d) **Read and categorize rows** from `log.parsedData?.rows || []`:
   - Rows with `conductorMatch.status !== 'resolved'` -> push to `skipped[]` with Hebrew reason:
     - ambiguous: `מנצח "${importedName}" - התאמה לא חד-משמעית`
     - unresolved/none: `מנצח "${importedName}" - לא נמצא`
   - Rows with `orchestraMatch.status === 'new'` and resolved conductor -> push to `toCreate[]`
   - Rows with `orchestraMatch.status === 'existing-updated'` -> push to `toUpdate[]`
   - Rows with `orchestraMatch.status === 'existing-no-change'` -> increment `noChangeCount`

e) **Bulk insert new orchestras** (`toCreate`):
   - For each row, build a document with: `tenantId`, `schoolYearId` (resolved), `name`, `type`, `subType` (|| null), `performanceLevel` (|| null), `conductorId` (from `row.conductorMatch.teacherId`), `memberIds: []`, `rehearsalIds: []`, `scheduleSlots` (via `buildScheduleSlots(row.schedule)`), `location: 'חדר 1'`, `ministryData` object with `coordinationHours`, `totalReportingHours`, `ministryUseCode`, `importedParticipantCount`, `isActive: true`, `createdAt: new Date()`, `updatedAt: new Date()`
   - Run each document through `validateOrchestra()`. If validation fails, push to `errors[]` with `{ row: row.row, name: row.name, error: validationError.message }` and skip that document
   - Call `orchestraCollection.insertMany(validDocs, { ordered: false })` if validDocs is non-empty
   - Wrap in try/catch: if `insertMany` throws (BulkWriteError), extract `err.writeErrors` array. Each writeError has an `index` property mapping back to the input array. Push failed rows to `errors[]`. Successfully inserted docs are still inserted (ordered: false).
   - Track `insertResult.insertedIds` for conductor linking

f) **Bulk update existing orchestras** (`toUpdate`):
   - Build `updateOps` array of `{ updateOne: { filter, update } }` objects
   - Filter: `{ _id: ObjectId.createFromHexString(row.orchestraMatch.orchestraId), tenantId }`
   - Update `$set`: `type`, `subType` (|| null), `performanceLevel` (|| null), `scheduleSlots` (via buildScheduleSlots), `'ministryData.coordinationHours'`, `'ministryData.totalReportingHours'`, `'ministryData.importedParticipantCount'`, `'ministryData.ministryUseCode'`, `updatedAt: new Date()`
   - CRITICAL: Do NOT include `memberIds` or `rehearsalIds` in $set -- these must be preserved
   - Also update `conductorId` if it differs from the existing one (it won't -- matching requires same conductorId -- but include it for completeness)
   - Call `orchestraCollection.bulkWrite(updateOps, { ordered: false })` if non-empty

g) **Conductor linking via bulkWrite:**
   - Build a `Map<string, Set<string>>` mapping `conductorId` -> set of `orchestraId` strings
   - For created orchestras: use `insertResult.insertedIds[i].toString()` as orchestraId
   - For updated orchestras: use `row.orchestraMatch.orchestraId`
   - Build `conductorOps` array of `{ updateOne: { filter: { _id: ObjectId.createFromHexString(conductorId), tenantId }, update: { $addToSet: { 'conducting.orchestraIds': { $each: [...orchestraIdSet] } } } } }`
   - Call `teacherCollection.bulkWrite(conductorOps, { ordered: false })` if non-empty

h) **Build affectedDocIds array:** Collect all created orchestra IDs (from insertResult.insertedIds) and updated orchestra IDs (from toUpdate rows' orchestraMatch.orchestraId)

i) **Build results object:**
   ```javascript
   {
     totalRows: rows.length,
     createdCount: (number of successfully inserted),
     updatedCount: toUpdate.length,
     skippedCount: skipped.length + noChangeCount,
     noChangeCount,
     errorCount: errors.length,
     skipped,     // [{ row, name, reason }]
     errors,      // [{ row, name, error }]
     affectedDocIds,
   }
   ```

j) **Update import_log:** Compute status using same pattern as teacher/student imports:
   ```javascript
   const totalSuccess = createdCount + updatedCount;
   const status = errors.length > 0 && totalSuccess > 0 ? 'partial' : errors.length > 0 ? 'failed' : 'completed';
   ```
   Update log with `{ status, results, affectedDocIds, completedAt: new Date() }`

k) **Wire the dispatcher:** Replace line 2333 (`throw new Error('ייבוא הרכבים יתמך בגרסה הבאה')`) with:
   ```javascript
   return await executeEnsembleImport(log, importLogCollection, tenantId, userId);
   ```

Use `?? null` (not `|| null`) for numeric fields like coordinationHours and totalReportingHours where 0 is a valid value.
